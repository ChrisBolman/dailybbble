API Endpoints

1. Popular Dribbble shots of a day

URI
{{ url_for('api_popular_day', _external=True, day='__date__') }}
Method
GET
Parameters
  • __date__: today or a date string in ISO 8601 pointing to a day. e.g. 2013-06-30.
Remarks
Provides popular shots sorted in descending order by likes in specified day. Results may be cached, and thus may be stale in a time window of 1 hour. Only first 20 results are returned.
